# lspec-tlc-pi

Adaptação do **TLC original** para o ecossistema PI.

## Princípio
- Comportamento/metodologia: **TLC original**
- Adaptação: apenas empacotamento PI (estrutura, comandos, instalação)

- `uninstall.sh`

curl -fsSL https://raw.githubusercontent.com/by-lua/lspec-tlc-pi/main/install.sh | bash
